## 📱 AI 프레젠테이션 기능 요약

### 4단계 사용 흐름 (정확한 UI 텍스트)

| 단계 | 액션 | 설정 항목 | 버튼명 |
|------|------|---------|--------|
| 1 | 주제 입력 또는 자료 업로드 | 슬라이드 수(6~30장), 청중, 발표 목적, 말투 | `[개요 만들기]` |
| 2 | AI 개요 검토 | 순서 변경, 추가/삭제 가능 | `[템플릿 선택]` |
| 3 | 템플릿 선택 | 교육/비즈니스/감성 중 선택 | - |
| 4 | 슬라이드 자동 생성 | 약 3분 소요, 에디터 편집 | `[생성하기]` |

### 핵심 용어 (반드시 정확히 사용)

| 용어 | 설명 |
|------|------|
| **miricle** (미리클) | AI 올인원 브랜드명 |
| **딱 맞는 페이지** | 슬라이드 레이아웃 자동 교체 기능 (1920×1080px 전용) |
| **개요 만들기** | Step 1 버튼 |
| **생성하기** | Step 4 버튼 |
| **AI 라이팅** | 이어쓰기, 요약하기, 다시쓰기, 맞춤법 |

### 요금 정보

| 항목 | 무료 | Pro |
|------|------|-----|
| 생성 횟수 | 월 5회 | 무제한 |
| AI 크레딧/일 | 200 | 1,000 |
| 프리미엠 템플릿 | 워터마크 | 전체 |
| 가격 | 무료 | 14,900원/월<br>또는 160,800원/연 |

### 내보내기 형식

PPTX (편집), PDF, PNG, JPG, MP4, GIF, 링크 공유

## ✅ 콘텐츠 작성 체크리스트

- [ ] 기능명: UI 텍스트 정확 (`[개요 만들기]`, `[생성하기]`)
- [ ] miricle 브랜드명 일관 사용
- [ ] H1: 첫 5단어 내 키워드 포함
- [ ] 메타 설명: 155~160자, CTA 포함
- [ ] 본문: 1,800~2,500자
- [ ] 구조: H2 3~4개, H3 각 2~3개
- [ ] 무료/Pro: 분명한 구분
- [ ] 내부링크: 3~5개
- [ ] Schema: FAQPage 또는 HowTo 적용
- [ ] 경쟁사 비교 피할 것 (미리캔버스 가치만 강조)

## 📝 SEO 페이지 구성 템플릿

```
1. Hero (H1 + 서브 + CTA)
2. Feature Cards (3개 - 주요 장점)
3. How-to 단계 (4단계 또는 5단계)
4. 차별화 기능 (교차 레이아웃)
5. 요금 비교표
6. FAQ (5~6개)
7. CTA 배너
```
